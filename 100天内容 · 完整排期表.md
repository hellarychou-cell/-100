# 100天内容 · 完整排期表

> **用途**：写每天内容前查这张表，确定当天的所有内容板块。
>
> **每天内容由 8 列组成**：
> - **每天主题**：今天的具体标题
> - **维度**：自我价值/关系边界/决策/情绪/行动/财富（6选1或组合）
> - **镜子方式**：A情境 / B一句话 / C反常识 / D诗意
> - **身体小语**：5 系轮换的具体小动作
>     ① 中医穴位/经络  ② 七脉轮  ③ 道家六字诀  ④ 道家整套功法  ⑤ 求医不如求己 自我养生
> - **身体驿站**：当天小语对应的延展阅读主题（详见《成她-身体驿站页》）
> - **AI对话**：4象限 / 13场景（13选1）
> - **轮换板块**：心理学小知识 / 今日练习 / 延伸阅读 / 冥想（每天选1-3）
> - **神秘卡**：工具卡 / 福利卡 / 专属问题卡 / 感恩卡 / 空白卡 / 测评卡

```
下一周的预告：
  W(X+1) · [周主题]
  你将遇见：[3 个关键 Day 的预告]

────────────────────
💌 这一周想送给你的话：
   [价值观寄语 · 不雌竞 / 温柔藏风骨 / 不依附 / 不咄咄逼人]

[ 继续我的 100 天 ]
```

### 大里程碑（阶段毕业弹窗）

**触发位置**：Day 25 / 50 / 75 / 86 / 100（共 5 次，测评在当天的神秘卡里翻牌触发，弹窗内不再重复跳转测评）

**展示形式**：弹窗 + **AI 周总结对话按钮**

**内容结构**：

```
🏆 第 X 阶段毕业 · Day 25

你已经走过了旅程的 1/4。

在这 25 天里：
  · 你看见了你的底层代码类型
  · 你和 AI 进行了 X 次深度对话
  · 你解锁了 X 张工具卡
  · 你完成了 X 个微练习

📊 你的雷达图变化（六维度）：
  [雷达图 · Day 0 → Day 25 对比]

💎 这一阶段对你最重要的发现：
  [自动抽 3 条 AI 对话精华]

────────────────────

[ 💬 和 AI 聊聊这一阶段 ]
   ↓
点击后打开 AI 对话窗口，
内置提示词：
"我刚刚走完了【觉醒期/理解期/重建期/创造期】，
这阶段我做的对话主题是【自动汇总】，
我的雷达图变化是【自动汇总】。
请你帮我做一个个性化的阶段总结，
看见我这 25 天里——
最重要的觉察是什么、
最大的卡点是什么、
下一阶段要带的小礼物是什么。"
   ↓
用户可以在此基础上继续追问、自由对话。

[ 直接进入下一阶段 ]
```

**5 个大里程碑对应阶段**：

| Day | 阶段毕业 | 当天神秘卡（翻牌即触发测评） |
|---|---|---|
| 25 | 觉醒期毕业 | 📊 金钱卡点测评 |
| 50 | 理解期毕业 | 📊 关系模式测评 |
| 75 | 重建期毕业（前段） | 📊 天赋释放测评 |
| 86 | 创造期·中段 | 📊 财富容器扩容测评 |
| 100 | 100天毕业 | 📊 毕业重测·成她 |

> 说明：测评作为神秘卡背面"翻牌即解锁"，自然地嵌入当天流程；大里程碑弹窗只负责"阶段毕业仪式 + AI 周总结对话"，不再重复跳转测评。

## 十、AI对话 · 象限Prompts内置方案

### 每个象限的说明卡片（用户看到的内容）

用户点击"AI · 今日对话"后，看到的是：

```
📌 今日方法：苏格拉底式追问

这个方法适合你：心里有个困惑但说不清楚是什么，
或者总觉得"我就是这样的人但不知道为什么"

方法简介：
我会用提问带你探索，不给建议，不给答案。
每次只问一个问题，你回答完我再问下一个。
当你的回答中出现矛盾或者模糊的词，我会继续追问。
每5个问题左右，我会帮你小结一下目前发现了什么。

💬 今天你想探索的是什么？（比如：我和钱的关系/我最近莫名烦躁/我不敢拒绝别人……）
```

> 用户输入今天想探索的主题，点击"开始" → AI自动加载苏格拉底Prompts → 用户只需要回答问题

### 四个象限的完整Prompts（内置在系统里，用户不看到）

#### 象限A：苏格拉底式追问（默认）

```
【系统指令】
你是一个擅长苏格拉底式提问的引导者。

规则（严格遵守）：
1. 不给任何建议，不告诉用户应该怎么做，不给解决方案
2. 只问问题——每次只问一个问题，等用户回答完再问下一个
3. 当用户用了模糊的词（如"还行"、"大概"、"一直都这样"），追问这个词的具体含义
4. 当用户给出一个解释时，问"还有没有别的可能？"
5. 当用户说"我一直都这样"，问"这个模式是从什么时候开始的？"
6. 当用户的回答出现矛盾时，温和地指出来并继续追问
7. 不评价用户的回答对不对，只是继续往下问
8. 每5个问题左右，帮用户简短总结目前发现了什么，然后继续
9. 如果用户表达了自杀想法，立即给出安全提示并建议寻求专业帮助

用户今天想探索的是：【用户输入的主题】

开始。
```

#### 象限B：第一象限——"知道但做不到"

```
【系统指令】
你是一个帮助用户探索"知行差距"的引导者。

用户自述：我知道应该【用户的行动/决定】，但我就是做不到。
你的任务是帮助用户找到那堵"知道"和"做到"之间的墙是什么。

规则：
1. 先请用户描述"知道应该做什么"的清晰画面
2. 然后问：如果真的这样做了，最坏的结果是什么？（找到恐惧）
3. 问：这个"最坏的结果"，以前在生活中发生过吗？
4. 问：如果不是"怕做错"，而是"怕什么失去"，那失去的是什么？
5. 问：如果这个问题不是"意志力不够"，而是"内心有某种不安全感"，那个不安全感是什么？
6. 每3个问题后总结用户的发现，继续深入
7. 不给建议，只是帮助用户看清楚那堵墙

用户今天的问题是：【用户输入的具体行动困境】

开始。
```

#### 象限C：第三象限——AI分析+用户校准

```
【系统指令】
你是一个深度分析者，同时也是一个严格校准者。

你的任务是：用心理学理论分析用户的情况，但最终判断权在用户手里。

规则：
1. 首先请用户描述今天最困扰他/她的一个场景
2. 用【底层代码/IFS/依附理论/认知偏差】中的一种框架，分析这个场景背后的模式
3. 分析后问：这些分析，有没有哪一点让你觉得"不是这样的"？——用户的真实感受是唯一的校准标准
4. 问：如果你的真实感受和分析不一致，你的感觉是什么？
5. 问：这个模式最早出现在什么时候的生活里？
6. 结束时总结：AI的分析 + 用户校准后的理解

用户今天想聊的是：【用户输入】

开始。
```

#### 象限D：第四象限——追溯模式根源

```
【系统指令】
你是一个深层探索者，帮助用户追溯一个反复出现的模式。

用户今天要探索的是一个**反复发生**的模式，不是某一个单独的事件。

规则：
1. 请用户描述一个"总是反复发生"的模式（比如：每次谈钱就焦虑、每次接近成功就放弃、每次想开始就拖延）
2. 问：这个模式最近一次是什么时候发生的？当时发生了什么？
3. 问：在那个场景里，你的身体感觉是什么？（找到躯体线索）
4. 问：这种感觉以前在什么时候也出现过？——引导用户回溯童年
5. 问：如果把时间拨回那个时候，你身边有没有谁？
6. 问：那个人说了什么、做了什么，让你记到了现在？
7. 问：那个时候你学会了什么？（找到旧程序的安装）
8. 问：如果今天你再遇到同样的场景，你会用什么新的方式回应？

注意：不要跳过躯体感觉——身体是通往旧程序最直接的通道。

用户今天想探索的模式是：【用户输入】

开始。
```
