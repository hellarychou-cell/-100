# 成她100 · 入口与展示页交付说明

## 本次交付范围

本次整理了公开主页、理念页、知识库展示页和测评流程退出入口。

## 页面路径

| 页面 | 路径 | 说明 |
|---|---|---|
| 公开主页 | `/` | 保留主视觉、登录/注册、测评入口、知识库入口 |
| 理念页 | `/philosophy` | 从主页轻入口进入，右上角 `×` 返回主页 |
| 知识库展示页 | `/knowledge` | 说明书式展示页，只展示 Day 1-25 |
| 测评资料页 | `/assessment/profile` | 右上角 `×` 返回主页 |
| 测评答题页 | `/assessment` | 右上角 `×` 返回主页 |
| 测评报告页 | `/assessment/result` | 保留保存报告，同时增加 `×` 返回主页 |
| 登录页 | `/auth?mode=login` | 可切换到注册，右上角 `×` 返回主页 |
| 注册页 | `/auth?mode=register` | 可切换到登录，右上角 `×` 返回主页 |
| 我的匣子 | `/treasure` | 标题直接作为跳转入口，不再折叠展开 |

## 首页调整

- 右上角只保留轻量的 `理念` 入口。
- 主页测评入口仍然进入 `/assessment/profile`。
- 主页知识库模块整体可点击，进入 `/knowledge`。
- 主页登录、注册分别进入 `/auth?mode=login` 和 `/auth?mode=register`。

## 交互与返回规则

- 独立展示页使用右上角 `×` 返回来源页或主页。
- 我的匣子不再使用折叠抽屉；`知识库`、`神秘卡`、`AI 总结`、`测评结果` 等标题直接跳转。
- 当前登录/注册仍是前端体验版，可完成路径跳转；真实账号保存与会员开通需要后续接入 Supabase Auth。

## 知识库展示规则

- 只展示 Day 1-25。
- Day 1-7 使用 `成她-Day1-7.md` 里的终版标题与关键句。
- Day 8-25 只展示阶段主题与“内容筹备中”提示，不伪造正式正文。

## 文案来源说明

- 理念页使用 `成她-说明书.md` 中的“成她宣言”。
- Day 1-7 标题与关键句使用 `成她-Day1-7.md` 的“第一周的 7 个镜子”。
- 未直接修改任何源文档。

## 验证

```bash
npm run lint
npm run build
```
