# 心理学工具箱

> 24 张可视化工具卡，每张对应一个具体的"旧程序卡点"，配套落地操作。
> 这是 100 天计划的"工具书层"——**Day 内容是给你看的，工具卡是给你用的**。

---

## 写作规范（必读）

所有工具卡必须遵循 [[工具卡写作规范]]——含统一头部模板、6 维度流派挂靠地图、文字风格准则（土象专业路线，与 Day 内容的水象情感路线分开）。

每张卡的标准结构：

```
🧰 心理学工具 · 当下即用（4 行信息条：理论依据 / 流派 / 适用 / 周期）
🪞 这张卡解决什么问题（3-5 个反问命中痛点）
🧠 核心原理（150-300 字 · 含专业术语 + 作者引用 + 机制描述）
🛠️ 怎么用（3 步落地 · 至少含 1 个梯度表/句式表/对照表）
⚠️ 重要提醒（澄清最常见误用）
💎 一句话总结（用户合上手机带走的那句）
```

**不再使用**：
- ❌ 🌰 一个生活例子（归 Day 内容的"她的故事"）
- ❌ 📌 在 100 天里出现在哪几天（破坏复用性）

## 工具与产品的连接方式

**用户视角**：在网站上每张卡可以独立查看、收藏、下载（带恬馨水印图卡）。完成 100 天后用户拥有"我的工具箱"——24 张随时可调用的心理工具。

**写作视角**：每张卡都用恬馨的 IP 语言改写过——不是百科式定义，是"用户能看懂、能记住、能用"的版本。

**差异化**：
- **6.1-6.4 是恬馨独家**——其他疗愈/心理 IP 没有这套财富容器视角
- **1.2 配得感四区** 是你的独创框架（金钱/关系/喜悦/休息）
- **3.4 60 分决策法** 是你的独创落地工具
