# 6.1 · RAS 网状激活系统

> **科学版的"吸引力法则"——不靠玄学，靠你的大脑滤镜怎么调**

## 🪞 这张卡解决什么问题

为什么有些机会身边的人都看见了，你看不见？
为什么你说"我赚不到钱"，钱真的不来？
为什么换了一种心态后，突然到处都是机会？
为什么同一个朋友圈，有人看到合作，你只看到攀比？

不是宇宙在听你说话——是你的大脑里有一个**滤镜**，只让你看见你"相信"的那一类信息。
这个滤镜叫 **RAS（网状激活系统）**——它一直在替你过滤现实，而你以为那就是现实。

---

## 🧠 核心原理

1949 年，意大利神经生理学家 Giuseppe Moruzzi 与 Horace Magoun 在论文《Brain stem reticular formation and activation of the EEG》中首次定位**脑干网状结构**对觉醒和注意力的调控作用——这套系统后来被命名为**网状激活系统**（Reticular Activating System，RAS）。

它做的事很简单：**在大脑每秒接收的约 1100 万 bit 信息里，决定哪 40 bit 进入你的意识**。

判断标准只有一个：**你反复在想的、相信的、关注的东西**。

这与 Donald Broadbent 在 1958 年《Perception and Communication》提出的**选择性注意过滤模型**（Selective Attention Filter Model）一致——人脑不可能处理所有感官输入，必须通过一个早期过滤器筛选"对自己重要"的信息进入工作记忆。

举一个最直观的例子：你刚买了一辆白色 Tesla，第二天上路，你会突然发现满街都是白色 Tesla。不是世界变了——是你的 RAS 把"白色 Tesla"加进了"重要"清单。

钱、机会、贵人也是一样的工作机制。

> 你的旧程序如果在说"赚钱很难"——RAS 就会自动过滤掉所有"赚钱很容易"的证据，只让你看见"果然又赚不到"的画面。
> 你以为你在客观看世界，其实你只看到了你"被允许看见"的那一部分。

**这就是为什么换了一个底层代码，整个世界看起来都不一样了。**——不是世界变了，是你的滤镜换了。

## 🛠️ 怎么用

### 第一步：识别你正在喂给 RAS 什么

打开手机备忘录，写下你每天对自己说**最多的 3 句话**——不思考，第一反应：

| 编号 | 你最常对自己说的话 |
|---|---|
| 1 | |
| 2 | |
| 3 | |

这 3 句话就是你 RAS 的"过滤指令"——它一直在帮你**找证据**证明这些话是对的。

---

### 第二步：把抽象指令换成具体画面

光说"我相信我能赚钱"没用——RAS 处理不了抽象的句子，它只认**具体的画面**。

| 旧指令（抽象） | 新指令（具体可见） |
|---|---|
| 我能赚钱 | 我看到客户主动来咨询我的价格 |
| 我够好 | 我看到有人在我的内容下说"被治愈了" |
| 我有贵人 | 我看到有人介绍机会给我，不求回报 |
| 我值得被爱 | 我看到他主动给我倒了一杯水 |
| 我能拥有 | 我看到我家里那个我犹豫了三个月才买的茶具 |

新指令越**具体、越像电影画面**，RAS 越能识别。

---

### 第三步：每天找 3 个"证据"喂给它

每天睡前，写下今天发生过的、**支持新指令的 3 件小事**——再小都行：

- 今天有 1 个人给我点赞了
- 今天我的内容被收藏了 1 次
- 今天我对自己说了"做得不错"
- 今天有客户多问了一个问题（说明她在认真看我的内容）

不需要"很大的好事"。RAS 只需要**累积同一类信号**——它就会调整滤镜。

**21 天为一个最小调整周期**：
- 前 7 天：你可能感觉没变化——这是 RAS 还在重建数据库
- 第 14 天：突然发现"咦最近怎么会有这种好事"
- 第 21 天：滤镜稳定下来——你看见的世界变了

---

## ⚠️ 重要提醒：不是"想就能成"

RAS 不是魔法——它**不能凭空创造**机会，只能让你**看见已经存在但你忽略了的**机会。

所以这个工具一定要配合**行动**：

```
看见了机会 → 行动
行动了 → 又给 RAS 喂了"我能行"的证据
证据多了 → 滤镜更宽
滤镜更宽 → 看见更多机会
```

这是一个正向循环。但只有你**先动**，循环才会启动。

> 不是"相信了就会发生"，是"相信 + 看见 + 行动 = 螺旋上升"。

---

## 💎 一句话总结

**你的大脑不是看到什么就相信什么——是相信什么才看见什么。
换一个相信，整个世界看起来都会不一样。**
