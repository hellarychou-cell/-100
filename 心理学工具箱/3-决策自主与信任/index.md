# 3 · 决策自主与信任

> **选择困难不是因为选项太多——是因为你没有把自己的尺子先立好**

---

## 这一组解决的核心问题

为什么你明明能力很强，关键时刻却总是不知道该选什么？
为什么你选了之后总是后悔，觉得"如果选另一个就好了"？
为什么你做决定的时候，脑子里最响的声音是"别人会怎么看"？
为什么你总是对"要不要做"想太多，对"怎么做"想太少？

不是信息不够，不够聪明，不够理性。
是你在**做决定之前，没有把自己的地图先画好**。

四张工具卡覆盖决策的四个关键节点：

```
3.1 内在 SWOT ──→ 决策前：我的卡点和优势全景图
        ↓
3.2 价值观排序卡 ──→ 决策中：拿什么做判断标准
        ↓
3.3 直觉 vs 恐惧驱动 ──→决策时：分辨身体智慧和旧程序
        ↓
3.4 60 分决策法 ──→ 决策启动：先拿到 60 分的答案
```

**用法**：
- **不知道自己想要什么**：从 3.1 开始（内在 SWOT 全景图）
- **两个选项都很好不知道选哪个**：从 3.2 开始（价值观排序）
- **面对机会想退缩不知道该不该**：从 3.3 开始（直觉分辨）
- **想太多不行动**：从 3.4 开始（60 分启动）
