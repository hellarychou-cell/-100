# 2 · 关系边界与自我主权

> **讨好型创业女性最缺的不是"会做事"——是"会守住做事时的自己"**

---

## 这一组解决的核心问题

为什么你能拿下客户却拿不回钱？
为什么你能搞定项目却搞不定一句"不行"？
为什么你能照顾所有人却照顾不了自己？

不是"能力不够"——是**边界系统**没建立。
讨好型创业女性的能量泄漏点 80% 集中在边界——不该应的应了、不该送的送了、不该忍的忍了。

四张工具卡覆盖边界的四个层次——**认知层 / 行为层 / 沟通层 / 结构层**：

```
2.1 课题分离 ──→ 认知层：这件事到底是谁的事
        ↓
2.2 拒绝肌肉训练 ──→ 行为层：把"说不"从禁区移出来
        ↓
2.3 非暴力沟通 NVC ──→ 沟通层：拒绝时怎么说不让对方炸
        ↓
2.4 边界四类 ──→ 结构层：身体/时间/情感/精神边界各自怎么守
```

**用法**：
- **不知道哪些事不是自己的责任**：从 2.1 开始（认知拔根）
- **明明想拒绝就是开不了口**：从 2.2 开始（行为脱敏）
- **能拒绝但每次拒绝完关系炸**：从 2.3 开始（沟通技术）
- **被各种"应该"压得透不过气**：从 2.4 开始（结构盘点）

## 这一组工具的差异化

**为什么走专业路线**——

1. **不教"沟通技巧"**：市面上的"高情商拒绝模板"只解决"对方的反应"，不解决"你身体里的应激"。这一组先做神经系统脱敏，再做语言技术。
2. **有梯度感**：从陌生人到伴侣，5 级递进——讨好型用户最常见的失败是"想直接挑战 5 级"，这一组明确告诉用户"先练 1 级"。
3. **挂靠真实流派**：行为治疗 + 神经心理学 + 个体心理学——不靠"大女主"叙事，靠 60 年的临床证据。
