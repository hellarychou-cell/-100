# 1.1 · 萨提亚冰山模型

> **水面上的"我不够好"只有一句，水面下藏着七层——看见哪一层，决定改变从哪一层开始**

---

| 项目 | 内容 |
|---|---|
| 理论依据 | 萨提亚冰山模型（Virginia Satir, 1972）+ 自体心理学·镜像理论（Heinz Kohut, 1971） |

| 适用 | "我不够好"反复出现 · 自我攻击 · 配得感低 · 做事之后仍不满足 |
| 周期 | 单次觉察工具 · 长期观察同一层的反复模式 |

---

## 🪞 这张卡解决什么问题

为什么你已经升职加薪了，心里那个"我还不够好"的声音还是关不掉？
为什么别人夸你，你第一反应是"他们只是客气"？
为什么你完成了所有清单，还是觉得自己什么都没做？

不是你不努力。
是你一直在**水面之上处理水面之下的问题**——改表面，不动根基。

每一个"我不够好"背后，至少藏着七层不同的来源——
**只有分清楚哪一层在响，改变才能真正发生。**

---

## 🧠 核心原理

Virginia Satir 在 1972 年出版的《家庭治疗》（*Family Systems Therapy*）中提出**冰山模型**（Iceberg Model）——一个人的外在行为只是冰山一角，水面之下藏着七个层面：

**行为**（Behavior）——你做了什么或没做什么
**应对方式**（Coping）——你用什么姿态面对（讨好 / 超理智 / 指责 / 打岔）
**感受**（Feeling）——那一刻的即时情绪（愤怒 / 悲伤 / 恐惧 / 喜悦）
**感受的感受**（Feeling about Feeling）——你对自己感受的评判（"我不该这么难受"）
**观点**（Perception / Belief）——你基于经历形成的信念（"我不够好"）
**期待**（Expectation）——你对自己 / 别人 / 世界的预期（"我必须完美才值得被爱"）
**渴望**（Yearning）——所有人共享的核心需求（被爱 / 被接纳 / 自由 / 认可）
**自我**（Self / Being）——"我是谁"的本质（通常被层层覆盖，难以触及）

在 Kohut 的**自体心理学**（Self Psychology, 1971）框架下，"我不够好"的根源往往在于**早期镜像体验的缺失**——当一个孩子在成长过程中反复接收的是"你不够好"而非"你很好"，这个信念会被内化，形成**代偿性夸大自体**（compensatory grandiose self）——表面上拼命努力证明自己，本质上是在喂养一个从未被满足的镜像需求。

>冰山的每一层都需要不同深度的干预：
> - 行为层 → 技能训练
> - 感受层 → 情绪命名与接纳
> - 观点层 → 认知重构（见1.3 ABCDE）
> - 渴望层 → 深层疗愈个案
> - 自我层 → 长期整合工作（IFS 等）

## 🛠️ 怎么用

### 第一步：定位你的"不够好"在哪一层

当下一次"我不够好"出现时，用这张**冰山探查表**逐层向下探：

| 冰山层 | 探查问题 | 你的答案 |
|---|---|---|
| **行为层** | 我做了什么/没做什么？ | |
| **应对方式** | 我用什么姿态应对——讨好？超理智？指责？ | |
| **感受层** | 那时我感受到什么情绪？ | |
| **观点层** | 我对自己说了什么话？ | |
| **期待层** | 我对自己/别人有什么期待？ | |
| **渴望层** | 我真正渴望的是什么？（被爱/被认可/自由/归属） | |
| **自我层** | 剥掉所有角色，我是谁？（最难回答，也最重要） | |

大多数人停在**观点层**（信念）就觉得找到了——但这往往不是根。探查到**渴望层**，才会发现"我不够好"其实是"我渴望被认可但没有被给够"的变形表达。

---

### 第二步：识别你的高频卡点层

连续记录 7 次"我不够好"事件，每次做完冰山探查后，标注**哪一层出现次数最多**：

| 高频层 | 意味着什么 | 对应工具卡 |
|---|---|---|
| 观点层（信念） | "我不够好"被当成事实，不是假设 | 1.3 ABCDE 认知重建 |
| 期待层 | 你对自己有不合理的完美要求 | 1.2 配得感四区 |
| 渴望层 | 核心需求长期未被满足 | 深度个案 / 疗愈 |
| 自我层 | 与真实自我失联太久 | IFS（4.2）/ 内在小孩冥想 |

**记录格式**（备忘录）：`[日期] 事件：[什么触发了] → 高频层：[哪层] → 下次用哪张卡`

---

### 第三步：针对不同层次的对症干预

| 层次 | 短期干预 | 长期工作 |
|---|---|---|
| 观点层 | 即时自我追问："这是事实还是观点？" | 1.3 ABCDE 重写 |
| 期待层 | 降维期待："完美→足够好" | 1.2 配得感四区 |
| 渴望层 | 承认渴望："我需要被认可，我值得被认可" | 深度个案 |
| 自我层 | 每天问自己一次："今天我喜欢自己什么？" | IFS内在整合（4.2） |

---

## ⚠️ 重要提醒：冰山不是用来"分析死"的

冰山模型最大的误用是——把它当成另一个脑内分析工具，越探越深，越探越停不下来。

**工具的目的是行动，不是分析。**

每次做完冰山探查，必须问自己一个问题：**"我今天愿意为渴望层做一件什么事？"**

哪怕只是一句"我今天允许自己休息"——那才是冰山真正的意义：
**从"看见"到"行动"，中间不需要完美，只需要一件小事。**

---

## 💎 一句话总结

**"我不够好"不是事实，是信号。
冰山模型帮你找到信号从哪里来——
看见它从哪层升起，改变就从哪层开始。**
