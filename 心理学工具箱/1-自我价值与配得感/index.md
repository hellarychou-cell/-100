# 1 · 自我价值与配得感

> **"我不够好"不是事实，是信号——冰山模型帮你找到信号从哪层升起**

---

## 这一组解决的核心问题

为什么你已经升职加薪了，心里那个"我不够好"的声音还是关不掉？
为什么别人夸你，你第一反应是"他们只是客气"？
为什么你完成了所有清单，还是觉得自己什么都没做？

不是你不努力。
是你在**水面之上处理水面之下的问题**——改表面，不动根基。

四张工具卡从"诊断"到"重建"的四层路径：

```
1.1 萨提亚冰山模型 ──→ 诊断层：找到"不够好"在哪一层响
        ↓
1.2 配得感四区 ──→ 定位层：找到伤区（金钱/关系/喜悦/休息）
        ↓
1.3 ABCDE 认知重建 ──→ 重写层：换掉那个错误的前提
        ↓
1.4 自我同情三步 ──→ 整合层：失败时不被自己打倒
```

**用法**：
- **不确定"不够好"在哪一层**：从 1.1 开始（冰山探查）
- **知道不够好但找不到根源**：从 1.2 开始（配得感四区定位）
- **自我攻击声音反复**：从 1.3 开始（认知链路重写）
- **犯错后陷入羞耻**：从 1.4 开始（自我同情）

## 这一组工具的差异化

**为什么这组工具是整个产品的基础**——

1. **"我不够好"是讨好型创业女性的核心旧程序**：所有其他维度（边界/决策/情绪/行动/财富）的卡点，都能在自我价值层找到根源。
2. **从诊断到重建有完整路径**：冰山诊断 → 四区定位 → 认知重写 → 自我同情——不是一个概念，是一条路。
3. **有身体感**：每张卡都连接到身体信号——不只改变思维，改变身体记忆。
