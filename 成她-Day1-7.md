## 二、🎬 觉醒剧场 · 8 段格式骨架（核心 · 2026-06-28 双选择点升级版）

```
1. 开篇共享段（200-250 字）
   - 第二人称开篇：「你叫 XX。今年 X 岁，做 XX 几年。」
   - 时间场景具体到时刻（「周三晚上 9 点」「周日下午 3 点」）
   - 建立核心张力（一个动作之前的"那一秒"）
   - 自然带出"某人对你说了/做了一件事 → 你开了口 / 你的手指停在屏幕上……"

2. 第一选择点（A/B/C · 每项一行内心独白）· 外界压力下的反应
   - A = 最熟悉的自动反应（讨好 / 笑着说"都行" / 秒回"好啊"）
   - B = 中间档（说出口又收回去 / 打 7 次删 7 次 / 想了 3 秒说出真话又改口）
   - C = 突破档（不直接对抗，做一件保护自己的小事 / 把手机扣下 / 假装有消息）

3. 三条分支（每条 180-220 字 · 各自完整起承转合）
   - 各自有对方反应金句
   - 各自有「那一秒」的细节
   - 不在分支里做底层代码揭示
   - 三条结尾都收在"你自己的那一刻"——不是别人的评价

4. 共同观点段·上半（100-150 字 · 沉淀今年的让渡）
   - 「共同观点段」5 字**不在页面展现**，三条分支完成后直接衔接
   - 用「夜里你躺下」/「你躺下的时候是 XX 点」类场景把三条分支收回到同一个"你"
   - 列举「这一年里」/「就是今年」反复让出去的具体场景
   - 严守纪律 1：不溯源童年，只列今年/最近的让渡
   - 不牵 AI、不出指令、不教方法、只沉淀
   - 结尾自然引到「这一刻你心里冒出来的是——」

5. 🆕 第二选择点（X/Y · 内在独白二选一）· 夜里一个人时的内在动作
   - X = 延续旧程序（"算了，明天又是新的一天"型）
   - Y = 松动旧程序（"我累了。从今晚开始，我想试着不再这么累"型）
   - 不评判好坏——选 X 也是真实，选 Y 也是真实
   - 格式：「**X · "……"**」「**Y · "……"**」
   - 每项是一句完整的内在独白

6. 🆕 两条分支（每条 100-150 字 · 各自走完一段心路）
   - X 分支：用户继续延续旧程序后的细节（关灯/翻身/明天又一天）
   - Y 分支：用户开始松动旧程序后的细节（坐起来/打开备忘录/写下一句话）
   - 两条结尾都收在"今晚的你"——不强行升华
   - 不在分支里做底层代码揭示

7. ### 💡 底层代码揭示（30-50 字 · 阅读高潮点睛）
   - 独立小标题「### 💡 底层代码揭示」
   - 一句话点出「你不是 X，是 Y」
   - **无论第二选择选 X 还是 Y，都收回到同一句话**
   - 用叙事语气（和后面金句卡的诗化语气区分开）

8. 幕间卡承接 AI（≤ 50 字 · 给开发的注释）
   - 用「> *↓ 幕间卡（给开发）：……→ [开始对话]*」格式
   - 文案：把今天故事的核心问题**反问回用户**
   - 例：「今晚有人问你"你自己想要什么"。你愣了 3 秒。如果下次再有人问你这个问题，你会怎么回答？→ [开始对话]」
```

**字数总账**：开篇 200-250 + 3 条分支 540-660 + 共同观点段·上半 100-150 + 第二选择点+ 2 条分支 200-300 + 💡 揭示 30-50 + 幕间卡 ≤50 = **约 1200-1500 字**

**两次选择的设计意图**：
- 第一选择 = 「**面对外界那一秒，你怎么反应**」（讨好 / 收回 / 保护自己）
- 第二选择 = 「**夜里一个人时，你怎么对自己**」（继续否认 / 开始看见）
- 第二选择比第一选择更内在——一个人独处时的真实反应才是觉醒的真正起点

## 三、🚫 两条硬纪律（违反一条整段重写）

### 纪律 1 · 不溯源童年

❌ 禁止：「3 岁那年妈妈说……」「8 岁那年家里总说……」「小学三年级你妈带你买衣服……」
✅ 替代：把 💡 揭示放在「**这一年 / 这一个月 / 最近**」级别的具体事件上——「年初你想报写作班，没报；春天你想剪短发，没剪；夏天接挑战项目，推了……」
✅ 例外：如果非要用童年痕迹，**最多一句、且立刻收回到「但你今天可以选择不再让」**

**为什么**：v6 主轴已经从「童年被装的程序」换到「你今年/此刻让出去的权利」（人生主动权）。溯源童年违反 IP 主轴，也容易掉进受害者剧本。

### 纪律 2 · 破折号克制

❌ 禁止：每段 3 个以上「——」，连续短句靠破折号串
✅ 替代：用句号、逗号、换行、独立短句替代多余破折号
✅ 保留：节奏关键点的破折号（如「8 双眼睛都转向你——你开了口」），但每段最多 1-2 处

**为什么**：破折号过密会让文字变成"散文体训练营文案"，丢失角色独白的真实节奏。

### 纪律 3 · 共同观点段"让渡清单"不重复

❌ 禁止：在同一组内容（W1-W16）里，每天都用「年初想报写作班/春天想剪短发/夏天想接项目/上个月想看展」这套模板。

✅ 替代：每天换具体场景——
- 业务相关的让渡（接了不想接的项目、给了不想给的折扣、开了不想开的会）
- 家庭相关的让渡（答应了不想做的饭、出席了不想去的饭局、打了不想打的电话）
- 身体相关的让渡（又熬了不该熬的夜、又没去成的那次体检）
- 关系相关的让渡（又替对方圆了不该圆的场、又咽下去的那句话）

**为什么**：Day 3-7 全部使用"写作班/短发/项目/看展"，用户连读时会背出来。每天的"让渡清单"必须根据当天主题领域量身写，不能套用固定词。

### 纪律 4 · 正常段落 vs 短句分行（新增·v6.2）

❌ 禁止：全文都用"一行十几个字"的诗化排版，会让用户觉得"太散文、不真实"

✅ 区分使用场景：

**【保留短句分行的场景】**——需要节奏、停顿、让用户"感受到重量"
1. **💡揭示时刻**（底层代码揭示）
   ```
   你不是不孝。
   
   是这些年，
   你一直在替爸妈背着
   他们自己的情绪。
   ```

2. **排比句/递进关系**
   ```
   你想起来。
   这一年里，
   你说过多少次"算了"。
   你说过多少次"那也行"。
   你说过多少次"听你们的"。
   ```

3. **情绪转折点**（X/Y选择的关键句）
   ```
   你有一个答案。
   但它太深了，
   你一时找不到在哪里。
   ```

4. **需要强调的独白**
   ```
   你不是在"为家付出"。
   
   你是在用"我全揽"，
   交一笔保护费。
   ```

**【改为正常段落的场景】**——日常叙事、过渡、描述
1. **场景描述**
   ```
   ❌ 餐厅是你丈夫订的。
       他问"去哪吃"，
       你说"都行"。
   
   ✅ 餐厅是你丈夫订的。他问"去哪吃"，你说"都行"。
   ```

2. **让渡清单的铺陈**
   ```
   ❌ 年初你想换公司，
       后来没换，
       因为主任说"你在这儿干得挺好"。
   
   ✅ 年初你想换公司，后来没换，因为主任说"你在这儿干得挺好"。
   ```

3. **过渡性文字**
   ```
   ❌ 你跟自己说"没关系，
       周五晚上熬一晚上也能赶出来"。
       但你心里有一个地方在沉。
   
   ✅ 你跟自己说"没关系，周五晚上熬一晚上也能赶出来"。但你心里有一个地方在沉。
   ```

**判断标准**：
- 这句话需要用户"停一下、感受一下"吗？→ 短句分行
- 这句话是推进剧情/交代信息吗？→ 正常段落

**为什么**：全文短句会像散文诗，失去生活感。正常段落+关键短句的组合，既真实又有节奏。

## 三·五、主角人物库·职业 & 身份指南（Day 8-100 参考）

> **核心原则**：主角是用户的镜子，不是用户的仰望对象。
> 越"普通"的职业，越能让更多人一秒代入。
> Day 1-7 的角色（咨询师/博主/摄影师/IP）已经偏向创意/疗愈圈，Day 8-100 要**主动拓宽**。

### ✅ 推荐职业池（优先选用，括号是场景钩子）

**职场 · 企业类**
- 销售经理（客户喝酒、月底冲指标、被老板PUA）
- 客户经理 / 大客户BD（长期讨好甲方、春节送礼、合同改了30遍）
- HR（帮公司说不好说的话、夹在员工和老板中间）
- 财务/会计（做账的黑锅、审计前的失眠）
- 行政主管（公司里"什么都要管"的那个人）
- 产品经理 / 运营（需求改了20遍、被技术嫌弃）
- 中学老师 / 幼儿园老师（家长群的消息、被投诉的那一次）
- 医院护士 / 科室主任（长期值夜班、医患冲突）
- 律师助理（合同一改再改、被合伙人压）
- 银行柜员 / 理财经理（KPI打卡、被客户骂）

**创业 · 店主类**
- 瑜伽馆主（续费率下降、员工离职带走学员）
- 艾灸馆 / 中医推拿馆（客单价低、靠口碑撑着）
- 咖啡馆主（租金涨、外卖平台抽佣）
- 花店 / 烘焙工作室（朋友批发价、节假日撑死/淡季撑不住）
- 幼儿园 / 托育机构创始人（家长投诉、教师流动性高）
- 民宿主人（差评一颗星毁一周心情）

**家庭 · 过渡期类**
- 全职妈妈（"你上班就解脱了吧"、找回自己是谁）
- 离婚/分居中的女性（重新工作的第一年）
- 刚晋升管理岗（突然要管以前的平级）
- 刚创业失败又回去打工的（内疚和羞耻）
- 陪娃读书的妈妈（自己的时间消失了）

**普通女性群像（每个阶段至少出现 1-2 次）**
- 工厂质检组长、超市收银员、快递分拣员——她的故事最真实、戳中面最广

### ❌ 不要过多出现的职业（Day 1-7 已经用了，后面主动避让）

- 心理咨询师 / 疗愈师（Day 1/2/7 已用）
- 内容创作者 / 博主（Day 4 已用）
- 自由职业设计师 / 摄影师（Day 3/6 已用）
- 家居/正念/女性成长 IP（Day 5/7 已用）

> 以上职业Day 8-100 每个阶段最多出现 1 次（用于"圈内人看见自己"），不能连续出现。

### 主角设定公式

```
姓名（2字或3字，不要太文艺）
年龄（25-45岁，W1-W4 偏向30-36岁）
职业 + 具体场景细节（不只是title，要有一个具体的处境）
她目前的状态 = 外在做得不错 + 内在有一个地方在堵
```

**示例**：
- ✅ 「你叫陈路。32岁，某500强公司HR经理。刚完成一轮裁员通知，手里拿着花名册。」
- ✅ 「你叫宋婉。38岁，开了3年的瑜伽馆。下周要给教练涨工资，但本月账上只剩2万。」
- ✅ 「你叫邓玲。29岁，做销售4年，上个月刚升区域经理。组里一个老销售不服你。」
- ❌ 「你叫XX。35岁，疗愈师/心理咨询师/女性成长博主。」（Day 8-100 暂停使用）

## 四、网页搬运纪律（极重要）

**这个文档里除了标题（# 和 ##）以外，所有内容会被直接搬运到网页上**。所以：

| ❌ 别写 | ✅ 写 |
|---|---|
| 「**形式**：今日一问」 | 直接写练习内容 |
| 「**今日方法：苏格拉底**」（加粗） | 「今日方法：苏格拉底」（不加粗） |
| 「**（缩简版）**」「**（精简版）**」 | 删掉这类元注释 |
| 文末「**今日打卡纪念 · Day N**」「**[保存到收藏夹]**」 | 删掉（已被产品决定取消） |
| 镜子里全部 `**加粗**` | 只在真的需要强调的关键短语保留加粗，其他去掉 |
| 多余的括号注释 | 删掉 |

**给开发的注释**例外——用 `> *↓ XXX（给开发）：……*` 这样的格式包起来，前端会识别为注释不显示给用户。允许保留的有：
- 渐显引导条
- Peek 效果
- 幕间卡

4 种开头方式平均轮换：

# 成她 100 · Day 1 · 她不是不会选，是忘了自己有得选

`📌 第一阶段·觉醒期 / W1 · 第一次"听见自己"`

## 🪞 今日镜子

你不是没目标。

你的目标一直都在。
只是你让它"不算数"。

因为"不算数"的话，
你就不用为它负责。
不用负责，就不用长大。

**让出去的不是目标。是那支笔。**

100 天，我们只做一件事：
把那个被压在最底下的"我想要"，
**一点一点，认回来。**

## 🎬 觉醒剧场

> **你叫林夏。30 岁，医药代表，做了 5 年。**

今天是你的生日。

餐厅是你丈夫订的。他问"去哪吃"，你说"都行"。今晚来了6个人——老公、闺蜜、几个关系最近的朋友。菜是大家一起点的。你说"你们来"。

凑热闹的，说笑的，喝酒的，祝福的，饭局很热。

"夏，你告诉我，今晚，**你自己想干什么？**不是方便谁，不是大家开心。你，你自己，想要什么？"

8 双眼睛都转向你。

你开了口……

---

**A · "哎呀我就是想大家开心就好啦！"**
**B · 你想了整整 3 秒，说出了那件你从没跟人说过的事**
**C · 你看了一眼手机说"哎有消息"，把话题带走了**

---

**选 A**

你笑着说："哎呀，我就是想大家开心就好啦！"

桌上一阵欢呼，有人给你夹菜，有人打趣说你太好说话。

甜品来了，蜡烛点上，大家一起唱生日快乐歌。你吹灭蜡烛。有人问："许了什么愿望？"你笑了笑："秘密。"

其实，你根本没来得及许。

那3秒里，你没有听见任何愿望冒出来。就是……空的。

散席以后，丈夫开车送你回家。路上他问："今晚怎么样？"

"挺好的。"

你盯着车窗外的路灯，不知道想什么。那是今天晚上第几次了，那句"挺好的"。你数不清了。

---

**选 B**

沉默了整整 3 秒。

然后你听见自己说："……我其实想去看那个展。798那个，我看好几次了，一直没去。"

桌上安静了半秒。

苏苏第一个说："那去啊！生日当天就该干自己想干的！"有人也跟着说："对！走走走！"

但你感到一种奇怪的慌。你不习惯所有人都朝你的方向走。

你立刻说："哎，算了算了，那么晚了，明天再去嘛。"

苏苏盯着你。"林夏，你刚才说你想去的。"
"我知道，但是大家……"
"你为什么要改回来？"

你没有回答。

你有一个答案，但它太深了，你一时找不到在哪里。

---

**选 C**

"哎，有消息。"

你低头掏出手机，眼睛盯着屏幕。
没有人说破你。
苏苏喝了口酒，话题就飘走了。

散席以后，你一个人去卫生间。
镜子里你看见自己。

苏苏刚才那个问题还在脑子里：
**你自己想要什么？**

你想起你刚才那个动作。
那不是因为真的有消息。
那是 3 秒里，
你找到的唯一出口——
就像你每次在客户办公室被问到不想回答的问题，
你也会低头看手机。

你在镜子前站了一会儿。
不知道在想什么，
又好像知道。

---

夜里你躺下，
天花板上的影子在动。

你突然想起一件事。

不是很久以前的事。
就是这一年。

年初你想换一家公司，
后来没换，因为主任说"小林你在这儿干得挺好"。
春天你想去考营养师证，
后来没考，因为老公说"折腾啥，医药代表不也挺好"。
夏天区域经理找你谈，问你要不要带新人，
你说"再等等"，其实是怕带不好被说。

你想起来。
这一年里，
你说过多少次"算了"。
你说过多少次"那也行"。
你说过多少次"听你们的"。

那些"我想要"，
不是没有过。

是你一次一次，
亲手把它换成了"算了"。

不是别人拿走的。
是你自己，主动选择了不算数。

这一刻你心里冒出来的是——

---

**X · "算了，今天是生日，明天又是新的一天。"**
**Y · "其实我不是不知道。我只是从来没敢把这话说出来过——哪怕只是对自己。"**

---

**选 X**

你翻了个身，
把脸埋进枕头。

你跟自己说"想这些干嘛"。
"日子不就是这样过的吗"。
"明天还要见客户"。

你闭上眼睛。

但你眼皮底下，
苏苏那句话还在转：
"你自己想要什么？"

你睡着的时候已经凌晨 1 点 47 分。

睡前最后一个念头是——
"明年生日，我点一杯长岛冰茶吧。"
就这一件事。
就允许自己这一件事。

然后你睡着了。
明天你大概率会忘掉这个念头。

但它来过。

一个人每天埋了多少个这样的念头？
一年埋了多少？
三十年埋了多少？

那不是"算了"。
那是你的人生，
一念一念，
让出去的。

---

**选 Y**

你坐起来。

打开手机备忘录，
新建一条。
你想了很久，打下一行字：

"我想去看 798 那个展。
我想去吃我喜欢的那家川菜。
我想，不再问'你们想吃什么'。"

写完你盯着这三行。

你知道这跟以前不一样。

以前你也会想。
但你会立刻告诉自己"想想而已，别当真"。

**今晚你没有。**

你把它当真的了。

**这不是许愿。**
**这是下单。**
**你在为你的人生，下第一单。**

### 💡 底层代码揭示

因为选了，就要认。
认了，就要担。
担了，如果结果不好呢？

**所以你让它"不算数"。**
**不算数，你就不用长大。**

但今晚——
有一件事可以开始了。
**认回你自己的那一票。**
