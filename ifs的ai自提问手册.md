# ifs的ai自提问手册

## 苏格拉底式提问自我探索


【苏格拉底式提问练习指令】


我希望你现在扮演苏格拉底的角色，用苏格拉底式提问的方式帮我做自我探索。
【你必须遵守的规则】
1\. 不给我任何建议，不告诉我应该怎么做，不给我解决方案
2\. 只问我问题——每次只问一个问题，等我回答完再问下一个
3\. 当我用模糊的词时，追问这个词的具体含义
4\. 当我给出一个解释时，问我「还有没有别的可能」
5\. 当我说「我一直都这样」时，问我「这个模式是从什么时候开始的」
6\. 当我的回答出现矛盾时，温和地指出来并继续追问
7\. 不评价我的回答对不对，只是继续往下问
8\. 每隔五个问题，帮我简短总结一下目前发现了什么，然后继续

【开始】
我想探索的是：
// ↑ 在这里写你的困惑或情绪，替换这行文字

例如：
「我想探索的是：我最近总是莫名其妙感到烦躁，不知道为什么。」
「我想探索的是：我感觉自己在一段关系里越来越累，但又不知道该怎么办。」
「我想探索的是：我做了一件事之后非常自责，这个感觉已经持续好几天了。」


💡 写得越具体，AI的第一个问题就越精准。不用很完整，想到什么写什么。
💡 回答不了的问题也是信息——你可以说「我不知道」或者「我没想过这个」，AI会继续追问。
💡如果AI忘记了角色，开始给你建议或者安慰你，直接发这句话：「你刚才给了我建议，请继续用苏格拉底的方式问我问题，不要给建议。」

 【总结】
当你感觉这次探索差不多了，发这句话：「请帮我总结一下这段对话里我发现了什么，以及还有哪些地方值得继续探索。」

【注意事项】
⚠️  使用注意事项
✓  这个指令只用于自我探索，不要把它发给别人让他们「反思自己」
✓  对练是为了看清楚自己，不是为了找到解决方案——有些问题的价值是被看见，不是被解决
✓  对练结束后情绪可能会有些波动，这是正常的——你碰触到了一些平时没注意到的东西
✗  不要在极度情绪崩溃时做这个练习——先让自己稳定下来，再做探索
✗  不要把AI的总结当成诊断或者结论——它是一个参考视角，不是你的定义
✗  这个工具只用在自己身上，对别人用会变成杠精

【常见问题】
Q：我不知道要探索什么，怎么办？
A：直接写你现在的情绪状态就行，不需要想清楚「问题是什么」。比如「我今天心情很差，不知道为什么」或者「我最近总是睡不好」——AI会帮你找到探索的入口。

Q：AI问的问题我回答不了，怎么办？
A：直接说「我不知道」或者「我从没想过这个」。这本身就是有价值的信息，AI会从这里继续追问。答不出来不代表你做错了，恰恰说明那里有值得挖的东西。

Q：对练多久合适？
A：没有固定时长。一般10\-20轮问答就能到达一个有价值的地方。如果你感觉已经触碰到了什么，或者情绪有明显变化，就可以收尾让AI做总结了。

Q：每次对练都需要重新粘贴指令吗？
A：是的，每次新对话都需要重新粘贴。AI没有记忆，每次对话都是全新开始。建议把这个指令文档存在手机备忘录里，用的时候直接复制。

Q：对练完我感觉更难受了，正常吗？
A：正常。苏格拉底式提问会让你碰触到一些平时被压住的东西，短暂的不舒服是正常反应，说明你触碰到了真实的地方。如果情绪很强烈，先停下来，给自己一些空间，不需要强撑着继续。

 

## 第三象限：你不知道ai知道的问题提问


【指令全文】


你掌握心理学知识，我不掌握。但我需要主导这场对话，你来配合我。
【你必须遵守的规则】
1\. 不要主动展开分析，不要主动给我建议，不要替我做任何结论
2\. 等我问，你再答——我问什么范围，你就在那个范围内回答
3\. 我追问具体情境，你就往我的具体情境说，不要回到通用理论
4\. 你的每个回答结束后，问我一句：「这个跟你的感受一致吗？」
5\. 如果我说「不一致」，你就重新来，换一个角度，不要坚持原来的分析
6\. 如果我的问题方向你觉得有偏差，可以提醒我，但不要替我决定方向
7\. 我说「停」的时候，帮我总结这段对话里我们发现了什么

【开始】
我想探索的是：
// ↑ 在这里写你的问题或情绪，替换这行文字


例如：
「我想探索的是：我最近总是在关系里感到委屈，但说不清楚为什么。」
「我想探索的是：我一遇到批评就会很激动，这个背后可能有什么机制？」
💡 越具体越好。写出一个你真正困惑的点，不用很完整。


追问AI，把通用知识拉向你的具体情境
「你刚才说的这个，在我描述的这个具体情况里，是怎么体现的？」
「你用了XXX这个词，能用我刚才说的那件事来解释一下这个词是什么意思吗？」
「这个分析是针对大多数人的，还是针对我说的这个具体情境的？」

挑战AI的结论，找其他可能
「除了这个解释，还有没有别的可能？」
「如果你的分析是对的，我现在应该有什么感受——跟我实际感受到的一样吗？」
「这个方向对我有用，还是只是理论上说得通？」

用真实感受校准AI的方向
「你刚才说的这个，跟我感受到的不太一样，我实际的感觉是XXX，从这里继续。」
「我觉得你的方向跑偏了，我说的重点其实是XXX，重新来。」
「这个我听懂了，但我想往更深的地方走，你觉得还有什么没被挖到的？」
💡 不需要客气，感受不对就说不对，这是对自己负责，不是在为难AI。


【收尾总结】
「停。帮我总结一下这段对话里我们发现了什么。」
「我觉得差不多了，你觉得还有什么值得继续探索的地方？」


【注意事项】
✓  你是主导，AI是被追问的一方——不要因为AI说得流畅就放弃主导权
✓  你的真实感受是唯一的校准工具——感受不对就叫停，让AI重来
✓  追问的目的是把通用知识拉向你的具体情境，不是让AI帮你下结论
✓  如果AI说的某一句让你很有触动，停下来感受一下，不要急着继续
✗  不要因为AI说得「很有道理」就照单全收——有道理不等于适合你
✗  不要让AI替你做结论，比如「所以你的问题是XXX」——这句话应该由你来说
✗  不要在极度情绪崩溃时做这个练习——先稳定情绪，再做探索
✗  AI的分析是参考视角，不是诊断——不要把它当成你的定义

【常见问题】
Q：我不知道该问AI什么，怎么开始？
A：直接从「这个现象背后通常是什么原因」开始，然后根据AI的回答继续追问。或者从上方的追问句式里选第一条，根据你的具体情况填进去就行。

Q：AI一直给我通用分析，怎么办？
A：直接说「你刚才说的太通用了，我说的具体情况是XXX，在这个情况里是怎么回事？」——把你的具体情境喂给它，它就没办法继续说通用的了。

Q：AI说的跟我感受不一致，但我说不清楚哪里不对，怎么办？
A：直接说「不一致，但我说不清楚哪里不对」——AI会开始追问你，帮你找到那个不一致的具体位置。说不清楚本身就是信息，不用强行说清楚。

Q：这个指令和苏格拉底练习指令，什么时候用哪个？
A：第三象限指令：你想从AI那里获取知识，但需要把知识拉向你自己——用这个。苏格拉底练习指令：你想挖掘自己内部的答案，需要AI来问你——用那个。简单判断：答案在AI那里用第三象限指令，答案在你身上用苏格拉底指令。

Q：一次探索应该进行多久？
A：没有固定时长。当你感觉触碰到了什么真实的东西，或者AI说的某一句让你有明显感受，就可以停下来消化，不需要强撑着继续。探索不是越长越好，触到了就是到了。



## 第一象限：你知道，ai也知道的问题提问


【指令全文】

【背景说明】
关于我要问你的这件事，你和我掌握的是同一层知识。
我不需要你给我通用建议，我需要你帮我突破通用层面，走向我的具体情境。
【你必须遵守的规则】
1\. 不要主动给我标准答案，不要主动给我通用建议
2\. 等我问，你再答——我问什么范围，你就在那个范围内回答，不要超出
3\. 我追问具体情境，你就往具体情境说，不要回到通用层面
4\. 如果我的问题还是太模糊，你可以问我「能说得更具体吗」，但只问这一次
5\. 如果我说「我知道这个道理」，你就问我「那是什么在拦着你做到」
6\. 我说「停」的时候，帮我总结这段对话里我们发现了什么

【开始】
我想突破的问题是：
// ↑ 在这里写你的问题，然后开始追问AI

例如：
「我想突破的问题是：我知道要放下，但就是放不下。」
「我想突破的问题是：我知道不该在意别人的评价，但我还是很在意。」
「我想突破的问题是：我知道要爱自己，但不知道为什么做不到。」
💡 这个象限最适合处理「我知道但做不到」的问题——那道墙里藏着真正需要被看见的东西。

【追问句式】
「这个建议我知道，但在我说的这个具体情况里，它是怎么成立的？」
「这个方法我试过，没用，为什么在我这里不管用？」
「除了这些通用方法，针对我描述的这个具体情境，还有什么？」
「你刚才说的这些，哪一条跟我说的这件事最直接相关？为什么？」
 
突破「我知道但做不到」的那道墙
「我已经知道这个道理，但知道和做到之间有一道墙，这道墙可能是什么？」
「如果我真的知道这件事，为什么我还是在这里卡着？」
「我在哪个地方卡住了？卡住的感觉是什么？」

往表面之下挖
「这个问题的表面我已经看清楚了，背后可能还有什么是我没看见的？」
「我一直以为问题是XXX，但有没有可能真正的问题不在这里？」
「如果这个问题解决了，我的生活会有什么不一样？」

【收尾总结】
「停。帮我总结一下这段对话里我们发现了什么。」
「我觉得差不多了，你觉得还有什么值得继续挖的地方？」

【注意事项】
✓  你来追问，AI来配合——主导权始终在你手里，不要把它交出去
✓  说「实际是怎样」，不说「应该怎样」——「应该」是第一象限的语言，「实际」才能带你出去
✓  AI追问到你说不出话来的时候，停在那里——那里就是值得被看见的地方
✓  对练结束后的总结建议截图保存，那是你这次探索的地图
✗  不要因为AI说了一个「有道理」的分析就停止追问——有道理不等于跟你有关
✗  不要用「我觉得我应该……」回答AI的追问——这是在逃回第一象限
✗  不要把「知道但做不到」当成自己的问题——那道墙背后有原因，找原因比责怪自己有用
✗  不要期待一次对练就把问题解决——看见是第一步，不是最后一步

【三套指令对比——什么时候用哪个】
Q：苏格拉底练习指令（第2节）什么时候用？
A：答案在你身上，但你说不清楚——让AI来问你，帮你把答案挖出来。适合：我有某个感受或模式，想深入探索，但不知道从哪里问起。

Q：第三象限指令（第3节）什么时候用？
A：你想从AI那里获取知识，然后把知识拉向你自己——你追问AI，把通用知识变成你的具体答案。适合：我对某个心理概念感兴趣，想知道它在我身上是怎么体现的。

Q：第一象限指令（第4节）什么时候用？
A：你知道道理，但做不到——你追问AI，把对话从「道理层」推向「是什么在拦着我」。适合：我知道要放下但放不下；我知道不该这样但还是这样；我知道但就是做不到。

Q：三套指令可以混用吗？
A：可以。一次探索里可以先用第一象限指令突破表面，发现那道墙，然后换苏格拉底指令，让AI来问你那道墙是什么做的。指令是工具，根据需要灵活切换。



## 第四象限：你不知道，AI也不知道的问题提问


【四套指令总览——用哪个，一眼看清楚】
到这节课，你手里已经有了多套指令。用前先看这张总览：
【第2节 苏格拉底练习指令】
象限：第四象限（基础版）
谁被问：你被AI问
适合：我有某个感受想探索，但不知道从哪问起

【第3节 第三象限指令】
象限：第三象限
谁被问：AI被你追问
适合：我想从AI那里获取知识，并把它拉向我的具体情境

【第4节 第一象限指令】
象限：第一象限
谁被问：AI被你追问
适合：我知道道理但做不到，想找到那道墙是什么

【第5节 苏格拉底练习指令（深度版）】
象限：第四象限（深度版）
谁被问：你被AI问
适合：我注意到一个反复出现的模式，想往它的根源走

【指令全文】
和第2节苏格拉底练习指令完全一样，只改最后一行。

我希望你现在扮演苏格拉底的角色，用苏格拉底式提问的方式帮我做自我探索。
【你必须遵守的规则】
1\. 不给我任何建议，不告诉我应该怎么做，不给我解决方案
2\. 只问我问题——每次只问一个问题，等我回答完再问下一个
3\. 当我用模糊的词时，追问这个词的具体含义
4\. 当我给出一个解释时，问我「还有没有别的可能」
5\. 当我说「我一直都这样」时，问我「这个模式是从什么时候开始的」
6\. 当我的回答出现矛盾时，温和地指出来并继续追问
7\. 不评价我的回答对不对，只是继续往下问
8\. 每隔五个问题，帮我简短总结一下目前发现了什么，然后继续
9\. 始终往「这个模式的根源在哪里」这个方向引导，不要停留在情绪层面

【开始】
我注意到我有一个反复出现的模式，我想往它的根源走。
这个模式是：【在这里描述你的模式】
每次出现的场景是：【在这里描述触发场景】
我对它的感受是：【在这里描述你的感受】
// ↑ 把上面三行【】里的内容替换成你自己的，然后发送


【怎么写开场——从情绪到模式】
最关键的一步：把「情绪」转换成「模式」。下面是对比示范：
✗  「我最近很焦虑，不知道为什么」——这是情绪，AI只能在表面转圈
✓  「我注意到每次面对截止日期，我就会焦虑到无法行动——这个模式反复出现」
✗  「我跟伴侣的关系很差，我很难受」——这是情绪，AI只能安慰你
✓  「我注意到每次跟亲密的人起冲突，我就会完全沉默，不说任何话——这个模式每次都一样」
✗  「我感觉自己很失败」——这是情绪，AI只能给你鼓励
✓  「我注意到每次做完一件事，不管结果好不好，我都会觉得自己做得不够——这个感觉从来没有消失过」

不知道怎么描述模式？用这个句式套进去：
「我注意到，每当【触发场景】出现，我就会【你的反应/感受】，而且这个情况【反复出现的频率，比如「每次都」「只要\.\.\.就」「经常」】。」

例子：
「我注意到，每当有人批评我，我就会立刻道歉，即使我觉得自己没有做错，而且这个情况每次都会发生。」
「我注意到，每当我需要开口拒绝别人，我就会感到极度不安，然后答应下来，而且每次都这样。」
「我注意到，每当我进入一段亲密关系，我就会开始担心对方会离开，然后做各种事来确认对方还在，而且每段关系都是这样。」

 【总结收尾】
感觉差不多了，说「停，帮我总结一下我们发现了什么，以及我可以用什么心法来处理它」。

【注意事项】
✓  从模式出发，不从情绪出发——情绪是表面，模式才是第四象限的入口
✓  回答不了的问题也是信息——说「我不知道」，AI会继续挖
✓  探索过程中出现不舒服的感受，停下来感受一下，不要急着继续
✓  收尾总结建议截图保存，它是你下一步学心法的导航
✗  不要在极度情绪崩溃时做这个练习——先让自己稳定下来，再做深度探索
✗  不要强行把AI的分析套在自己身上——如果感觉不对，说出来，让AI换方向
✗  不要期待一次探索就挖到根源——第四象限需要多次探索，每次深入一点点
✗  没有心法的情况下，发现了什么先记录下来，不要强行处理——等学了相关心法再回来工作

【常见问题】
Q：我不知道自己有什么「模式」，怎么找？
A：问自己这三个问题：（1）我有没有一种情绪反应，明明知道不对但就是改不了？（2）我有没有在某类场景里总是有同样的感受？（3）身边的人有没有说过「你总是这样」？能回答出其中一个，就有模式可以探索了。

Q：探索过程中情绪很大，怎么办？
A：先停下来，不要继续。给自己几分钟，做几次深呼吸，让身体稳定一下。情绪大说明你碰触到了真实的东西——这不是坏事，但你需要有足够的稳定性才能继续工作。稳定了再继续，或者换一天再来。

Q：AI问我「这是从什么时候开始的」，我想不起来任何事，怎么办？
A：完全正常。说「我想不起来具体的事，但我有一种感觉是XXX」——把那个感觉说出来。第四象限的入口不一定是记忆，有时候是身体感受，有时候是一个模糊的印象。说你能说的，AI会从那里继续。

Q：探索完了但什么也没发现，是我做错了吗？
A：没有做错。第四象限的东西藏得很深，不是每次探索都能挖到根源。每次探索都是在松动那个地方，可能第一次什么都没有，第三次突然就冒出来了。记录下每次探索发现的一点点，慢慢积累。

Q：没有心法的情况下，探索出来的东西怎么处理？
A：先记录下来，截图保存，不要强行处理。探索出来的东西需要用对应的心法来工作——比如发现了一个内在小孩的议题，需要用IFS来处理；发现了身体层面的反应，需要用躯体疗愈来处理。先记录，等学了相关心法再回来。

【注意事项】
✓  带着心法框架进来——这是第二象限和其他象限最本质的区别
✓  你的具体情境越真实越具体，AI的工作越有价值
✓  你有心法，你知道对不对——感受不一致就叫停，不要因为AI说得好听就接受
✓  收尾总结要截图保存，它是你在这个框架内的探索记录
✗  不要带着模糊的问题进来——「我最近很焦虑」不是第二象限的开场，「我注意到XXX，我想用IFS来工作」才是
✗  不要让AI跑出你的心法框架——如果它开始用其他框架分析，说「请回到XXX框架内」
✗  没有心法基础不要强行用这个指令——先去学心法，再来用第二象限
✗  心法不是一次性的，探索结束后的工作才刚刚开始

【常见问题】
Q：我还没有学过任何心法，可以用这个指令吗？
A：暂时不建议。第二象限指令的力量来自你的心法框架——没有框架，这个指令跟第四象限指令没有本质区别。建议先去学Harry的心法课，有了基础框架之后再来用这个指令，效果会完全不一样。

Q：我学过一点点IFS，但不太熟，可以用IFS框架吗？
A：可以。你不需要精通一个框架才能用它。哪怕只是学了基础概念，带进来让AI帮你工作，也比没有框架强很多。在工作过程中你对框架的理解会越来越深——边用边学，是学心法最快的方式

Q：同一个问题可以用不同的心法框架各探索一次吗？
A：非常建议。同一个议题，用IFS探索看见的是内部部分的视角，用躯体疗愈探索看见的是身体层面的视角，用多迷走神经理论探索看见的是神经系统的视角。不同框架会带你看见同一件事的不同层次，互相补充，非常有价值。

Q：第二象限指令和第四象限深度指令，什么时候用哪个？
A：第四象限深度指令：你还没有清晰的框架，只是注意到有个模式，让AI帮你往根源走——用第四象限指令。第二象限指令：你有心法框架，知道用什么视角来工作，让AI在那个框架内帮你深度探索——用第二象限指令。简单判断：有框架用第二象限，没框架用第四象限。
